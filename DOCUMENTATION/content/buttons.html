Buttons

This theme has a couple of types of buttons, from plain buttons with just text to buttons with icons. The default buttonss use the the 10px size icons from our custom made icon set.

Text only

Basic text buttons with no icons.

<a href="#" class="button-text">Text button</a>
<button class="button-text">Imagelist</button>
<input type="submit" value="Imagelist" class="button-text">

Icon + Text

Basic buttons with a 10px icon and text.

<a href="#" class="button-icon-text"><span class="imagelist-10 plix-10"></span> Imagelist</a>

Text + Icon

Basic buttons with text and a 10px icon.

<a href="#" class="button-text-icon">Imagelist <span class="imagelist-10 plix-10"></span></a>

Button with indicator

Basic text buttons with indicators.

<a href="#" class="button-text">Imagelist <p>23</p></a>

Split button - Icon + Text

Split buttons with a 10px icon and text. These buttons cant be changed in size or color.

<div class="button-split">
    <a href="#"><span class="download-10 plix-10"></span></a>
    <a href="#">Download</a>
</div>

Split button - Text + Icon

Split buttons with text and a 10px icon. These buttons cant be changed in size or color.

<div class="button-split-rev">
    <a href="#">Download</a>
    <a href="#"><span class="download-10 plix-10"></span></a>
</div>

Button sizes

The buttons come in 3 sizes(24px, 30px, 40px and 50px). The size class can be used for most button types.

<a href="#" class="button-text">Imagelist</a>
<a href="#" class="button-text button-large">Download</a>
<a href="#" class="button-text button-big">Download</a>
