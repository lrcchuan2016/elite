Icons

This theme has it's own custom made icon set called "plix", this icon set come's in 3 sizes, 10px, 16px and 32px. These custom made icons come in sprite and single png images(default: sprite), and yes the source is also included(organized layerd PSD). A complete list of the icon name's can be found in the main documentation.
